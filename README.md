## Setup Instructions
```bash
# Clone repository
git clone git@github.com:SourabhaKK/PAI-A1FT-Solo-Assignment.git

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
